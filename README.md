# Cognitive Career & Job Recommendation System

## Overview
A modern web application providing intelligent career guidance powered by Cognitive and Explainable AI technology.

## Features
- **AI Recommendations**: Personalized career and job recommendations using advanced artificial intelligence algorithms
- **Explainable AI**: Transparent AI explanations helping users understand recommendation reasoning  
- **Adaptive Learning**: System that learns and adapts to user preferences and feedback
- **Skill Gap Analysis**: Intelligent analysis identifying missing skills with improvement suggestions

## Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **Design**: Responsive design with Inter font family and Font Awesome icons
- **Deployment**: Optimized for Vercel deployment
- **Version Control**: Git with GitHub integration

## Project Structure
```
jobs/
├── frontend/           # All frontend assets
│   ├── index.html     # Main home page
│   ├── auth.html      # Authentication page
│   ├── style.css      # Main styles
│   ├── auth-style.css # Authentication styles
│   ├── script.js      # Main JavaScript
│   └── auth-script.js # Authentication logic
├── vercel.json        # Deployment configuration
└── README.md          # Project documentation
```

## Getting Started
1. Clone the repository
2. Open `frontend/index.html` in your browser
3. For development, serve the files using a local server

## Deployment
The project is configured for easy deployment on Vercel using the included `vercel.json` configuration.
